# 对标拆解

> 拆解值得学习的账号 / 爆款，提炼**可复用**的打法。调研用 agent-reach（见 `../AGENTS.md`「调研」章）。
> 每拆一个，按下面模板记一条；重点是「可复用点」，不是复述对方内容。

## 模板（复制下面这段，填一条）

### @{账号 / 博主名} — {一句话定位}

- **平台**：
- **拆解对象**：某条爆款 / 整个账号（用 agent-reach 搜，记线索不记死链接）
- **数据**：粉丝 / 赞藏评 / 评论区氛围（抓得到就记）
- **选题角度**：做对了什么选题？切的是哪个痛点？
- **标题 / 封面套路**：什么钩子？
- **内容结构**：怎么开头、怎么推进、怎么收？
- **可复用点**（最重要）：哪些能搬到我这里？怎么改造成我的调性？
- **别学的**：哪些和我的调性 / 铁律冲突？
